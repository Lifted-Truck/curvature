# CHECKPOINT 2 audition — flow as musical gesture

All files rendered through the real C++ engine (`render_offline` drives the
same GeometryService + ModalVoice path the plugin's threads use). But the
checkpoint question — *does flow sound like phrasing?* — is best answered by
playing the plugin live: Kick, then hold a chord with Flow=Relax; flip to
Sharpen mid-phrase; ride Flow Rate like a mod wheel.

- `icosphere_kick_relax.wav` — conformal kick at t=0, forward flow at rate
  0.6 while ringing: the geometric release envelope. Spectrum starts bent
  (~3x sharp fundamental), breathes home to 220 Hz.
- `torus_golden_kick_relax.wav` — same gesture on the golden lattice.
- `genus2_kick_relax.wav` — same gesture on the dense genus-2 spectrum.
- `icosphere_sharpen.wav` / `genus2_sharpen.wav` — reverse flow from the
  base metric: curvature concentration pushes the spectrum away from the
  attractor until the |u| clamps bound it. The implausible direction.
- `genus2_bowed_relax.wav` — bow excitation riding a relaxing metric:
  drone whose interior structure migrates.
- `torus_string_bowed_sharpen.wav` — the 8:1 harmonic torus bowed while
  sharpening: a string un-stringing itself.

Listen for: motion that reads as *one object changing* (claim 2 carried into
the engine), RELAX as release/resolution, SHARPEN as tension. Flow-rate
ranges and kick amplitude are first-ear-pass defaults — retune at will.
